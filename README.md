# ArrowPrecisionEvaluator

A SwiftUI prototype for evaluating arrow hit dispersion from colored marks on a target image.

## Current MVP Skeleton
- Navigation flow is implemented
- Manual calibration rectangle input
- Target point selection
- Color preset selection
- Marker review flow
- Basic analysis metrics calculation

## Planned Features
- Real camera capture
- Perspective correction
- HSV color segmentation
- Marker centroid detection
- CSV export
- Session persistence

## Tech Stack
- SwiftUI
- Foundation
- UIKit
